Mis on HTML?

HTML ehk HyperText Markup Language on veebilehtede struktureerimisekeel. See ei ole programmeerimiskeel,

    vaid märgendikeel, mis määrab ära, kuidas sisu (nt tekst, pildid, lingid, tabelid jms) veebilehel kuvatakse.

HTML-i põhiülesanded:

  • Määrata, mis tüüpi sisu on (nt pealkiri, lõik, link, pilt).
  • Struktureerida sisu loogiliselt (nt päised, jalused, sektsioonid).

CSS

Mis on CSS?

    CSS tähendab Cascading Style Sheets ehk Kaskaadstiililehed. See on keel, mida kasutatakse veebilehtede kujundamiseks — ehk siis selleks, kuidas HTML-iga loodud sisu välja näeb. CSS määrab värvid, fondid, paigutuse, ruumid, animatsioonid ja palju muud. Lihtsalt öeldes: HTML loob veebilehe sisu ja struktuuri (nt pealkirjad, pildid, lõigud). CSS ütleb, kuidas see sisu välja näeb — milline on tekst, kus elemendid asuvad, mis värvi on taust jne.
